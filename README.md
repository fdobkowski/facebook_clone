## Introduction
This app is a social media website highly inspired by facebook.com.
This website allows you to make an account, create posts, add friends and much more.

- resource API:

  - express.js - v4.18.1
- cache:
  - redis - v6.2.6
- websocket:
  - socket.io - v4.5.1

##### Frontend:
- framework:
  - ReactJS - v18.2.0
- state manager:
  - REDUX - v4.2.0
